# Beyond Work 3분 유튜브 튜토리얼 원고

## 영상 제목
Beyond Work 사용법: 하루 업무와 시간을 AI로 정리하는 방법

## 영상 목적
처음 보는 사용자가 Beyond Work의 핵심 가치를 3분 안에 이해하고, 바로 일간 일정 화면에서 우선업무와 시간표를 작성해보게 만드는 것.

## 핵심 메시지
Beyond Work는 단순한 할 일 목록이 아니라, 오늘의 우선업무, 시간별 일정, 반복업무, 프로젝트, 자금, 건강 리듬을 한 화면에서 연결해 실행력을 높이는 AI 플래너입니다.

## 0:00-0:20 문제 제기
화면: 스마트폰 또는 PC 화면에 흩어진 메모, 캘린더, 할 일 목록, 자금 메모를 빠르게 보여줍니다.

내레이션:
“일정은 캘린더에 있고, 업무는 메모에 있고, 중요한 일은 머릿속에 남아 있는 경우가 많습니다. 그래서 하루가 바쁜데도 정작 중요한 일은 밀리기 쉽습니다.”

자막:
일정, 업무, 프로젝트, 자금 흐름이 흩어져 있다면?

## 0:20-0:40 Beyond Work 소개
화면: Beyond Work 메인 화면. 일간 일정이 보이는 장면.

내레이션:
“Beyond Work는 오늘 해야 할 일과 시간표를 한 화면에서 정리하고, AI가 사용자의 목표와 패턴을 참고해 실행 가능한 다음 행동을 제안하는 플래너입니다.”

자막:
오늘의 우선업무 + 시간별 일정 + AI 코칭

## 0:40-1:15 일간 일정 사용법
화면: 오늘의 우선업무에 A/B/C 업무를 입력하고 체크하는 장면.

내레이션:
“먼저 오늘의 우선업무를 적습니다. 가장 중요한 일은 A, 중요한 일은 B, 가능하면 처리할 일은 C로 둡니다. 완료하면 체크하고, 진행 중이거나 위임, 연기할 일도 간단히 표시할 수 있습니다.”

화면: 반복 우선업무를 열고 요일 체크를 선택하는 장면.

내레이션:
“매일 반복되는 일도 원하는 요일만 선택해 자동으로 올라오게 할 수 있습니다.”

자막:
A/B/C 우선순위, 완료 체크, 반복업무

## 1:15-1:50 시간별 일정 사용법
화면: 시간별 일정. 30분/1시간 버튼 전환. 현재 시간대 표시.

내레이션:
“시간별 일정은 30분 단위와 1시간 단위를 선택할 수 있습니다. 변경한 시간 단위는 그날부터 적용되고, 이전 날짜의 기록은 그대로 보존됩니다. 현재 시간대는 별도로 표시되어 지금 어디에 있는지 바로 알 수 있습니다.”

화면: 일정 병합 버튼으로 긴 일정을 만드는 장면.

내레이션:
“긴 일정은 칸을 합쳐 한 덩어리로 기록할 수 있습니다.”

자막:
30분/1시간 전환, 현재 시간 표시, 일정 병합

## 1:50-2:20 AI 추천과 오늘의 코칭
화면: 우선업무 AI 추천 버튼 클릭. 추천 업무 팝업.

내레이션:
“AI 추천을 누르면 사용자의 목표, 현재 과제, 건강 리듬, 이월업무, 일정 상태를 참고해 오늘 넣을 만한 업무를 제안합니다.”

화면: 우측 말풍선 코칭 페이지.

내레이션:
“상단의 말풍선은 오늘의 총괄 비서 역할을 합니다. 업무가 과밀한지, 시간이 비어 있는지, 건강과 회복을 고려해야 하는지 알려줍니다.”

자막:
개인화된 업무 추천과 총괄 코칭

## 2:20-2:45 주간, 월간, 프로젝트, 자금, 시트
화면: 주간 일정, 월간 달력, 프로젝트, 자금, 시트 섹션을 빠르게 전환.

내레이션:
“주간과 월간 흐름을 보고, 프로젝트와 자금 이슈도 함께 관리할 수 있습니다. 커스텀 시트는 Numbers처럼 필요한 양식을 직접 만들어 쓰는 공간입니다.”

자막:
주간 계획, 월간 보기, 프로젝트, 자금, 커스텀 시트

## 2:45-3:00 마무리
화면: 일간 일정 화면으로 돌아와 완료 체크가 올라가는 장면.

내레이션:
“Beyond Work의 목표는 더 많은 일을 적는 것이 아니라, 오늘 정말 중요한 일을 실행하게 돕는 것입니다. 오늘의 한 칸부터 시작해보세요.”

자막:
오늘의 실행을 더 선명하게.

## 촬영 체크리스트
- 데스크톱 classic mode 화면 1컷
- 모바일 ceo mode 화면 1컷
- 오늘의 우선업무 입력 장면
- 반복 우선업무 요일 선택 장면
- 시간별 일정 30분/1시간 전환 장면
- 현재 시간대 표시 장면
- AI 추천 팝업 장면
- 오늘의 코칭 페이지 장면
- 프로젝트/자금/시트 섹션 빠른 전환 장면

## 화면 녹화 순서
1. 로그인 후 일간 일정 화면 진입
2. 오늘의 우선업무에 A/B/C 업무 입력
3. 완료 체크, 진행 중 점 표시, 연기 날짜 지정 장면 녹화
4. 반복 우선업무 열기
5. 매일 선택 후 요일 체크박스 조정
6. 시간별 일정에서 30분/1시간 전환
7. 현재 시간대가 표시된 행 보여주기
8. 일정 병합 버튼으로 긴 일정 만들기
9. AI 추천 버튼 클릭 후 추천 업무 선택
10. 우측 말풍선 코칭 페이지 열기
11. 주간, 월간, 프로젝트, 자금, 시트 메뉴를 빠르게 전환
12. 다시 일간 일정으로 돌아와 완료 숫자 확인

## 자막 세트
- 흩어진 업무와 일정을 한 화면으로
- 오늘의 우선업무를 A/B/C로 정리
- 반복업무는 필요한 요일에만 자동 생성
- 시간표는 30분 또는 1시간 단위로
- 현재 시간대가 바로 보이는 일정표
- AI가 목표, 이월업무, 건강 리듬을 참고
- 프로젝트, 자금, 시트까지 연결
- 오늘의 실행을 더 선명하게

4. Beyond Work 3분 사용법

Beyond Work는 오늘의 우선업무, 시간별 일정, 반복업무, 프로젝트, 자금관리, 커스텀 시트를 한 곳에서 관리하는 AI 플래너입니다. 이 영상에서는 처음 사용하는 분도 바로 따라 할 수 있도록 일간 일정 작성, AI 추천, 시간표 관리, 주간/월간/프로젝트 기능을 3분 안에 소개합니다.
