# Beyond Work 사용자 매뉴얼 구성안

## 1. Beyond Work란?
Beyond Work는 업무, 일정, 프로젝트, 자금, 개인 정보, 건강 리듬을 한 곳에 연결해 오늘의 실행을 돕는 AI 플래너입니다.

## 2. 기본 화면 이해
- Classic mode: 넓은 화면에서 전체 섹션을 한눈에 보는 모드
- CEO mode: 스마트폰과 좁은 화면에 최적화된 집중 모드
- 일간 일정: 오늘의 우선업무와 시간별 일정 중심
- 말풍선 코칭: 오늘의 총괄 AI 비서

## 3. 오늘의 우선업무
- A: 오늘 가장 중요한 결과
- B: 중요하지만 유연한 업무
- C: 가능하면 처리할 업무
- 체크: 완료
- 점: 진행 중
- 위임: 담당자 입력
- 연기: 날짜 지정

## 4. 반복 우선업무
- 매일: 일~토 중 필요한 요일 복수 선택
- 매주: 특정 요일 반복
- 매월: 특정 일자 반복
- 매년: 특정 월/일 반복
- 삭제 시 이전 기록은 유지하고 이후 일정만 중단

## 5. 시간별 일정
- 30분 단위: 촘촘한 일정 관리
- 1시간 단위: 하루 전체 흐름 관리
- 현재 시간대 표시
- 일정 병합/분리
- AI 일정 배분

## 6. AI 추천과 코칭
AI는 다음 정보를 참고합니다.
- 사용자 역할과 목표
- 현재 과제
- 건강 상태와 운동 목표
- 이월업무
- 오늘 일정
- 최근 완료 흐름

주의:
건강 관련 코칭은 의료 진단이 아니라 일정관리와 활동 강도 조절을 돕는 참고 기능입니다.

## 7. 주간/월간
- 주간 일정에서 한 주의 흐름 확인
- 월간 달력에서 공휴일, 음력, 기념일 확인
- 날짜 선택으로 일간 일정 이동

## 8. 프로젝트
- 프로젝트 목록
- 프로젝트 세부내용
- 프로젝트별 자금 시뮬레이션
- 좌우 스와이프로 목록과 세부내용 이동

## 9. 자금/용돈
- 월별 자금 체크
- 수입/지출/이자/카드대금 관리
- 반복지출 종료기한 설정
- 필요한 경우 우선업무에 반영

## 10. 커스텀 시트
- 필요한 양식을 직접 생성
- 템플릿 선택
- 셀 입력, 체크박스, 숫자, 날짜, 통화
- 행/열 크기 조정
- 제목행/제목열 고정

## 11. 보안
- 일정 시간 후 블라인드 처리
- 비밀번호/생체인증 기반 잠금
- 스마트폰에서는 불필요한 블라인드 기능 최소화

## 12. 추천 사용 루틴
아침:
- 데일리 프랭클린 노트 확인
- 오늘의 우선업무 3개 작성
- A업무를 시간표에 배치

오전:
- 가장 중요한 A업무 먼저 실행

오후:
- 이월업무와 위임/연기 판단

마감:
- 완료 체크
- 메모 AI 질문으로 회고
- 내일 첫 행동 하나 기록

## 13. 처음 사용자 빠른 시작
1. 로그인합니다.
2. 설정에서 사용자 정보를 간단히 입력합니다.
3. 일간 일정으로 이동합니다.
4. 오늘의 우선업무 A/B/C를 하나씩 적습니다.
5. 시간별 일정에 A업무를 배치합니다.
6. AI 추천 버튼을 눌러 빠진 업무가 있는지 확인합니다.
7. 말풍선 코칭을 열어 오늘의 조언을 확인합니다.

## 14. 자주 묻는 질문

### Q. A/B/C는 어떻게 구분하나요?
A는 오늘 결과를 바꾸는 일, B는 중요한 보조 업무, C는 가능하면 처리할 일입니다.

### Q. 완료하지 못한 업무는 어떻게 되나요?
미완료 업무는 다음 날로 이월되어 다시 판단할 수 있습니다. 완료, 연기, 위임, 취소 중 하나로 정리하면 됩니다.

### Q. 반복 우선업무는 어떻게 쓰나요?
반복 우선업무에서 주기를 선택합니다. 매일을 선택하면 일~토 중 필요한 요일만 체크할 수 있습니다.

### Q. 시간표 30분과 1시간은 언제 쓰나요?
일정이 촘촘한 날은 30분 단위가 좋고, 하루 전체 흐름을 크게 볼 때는 1시간 단위가 좋습니다.

### Q. AI 코칭은 어떤 정보를 참고하나요?
사용자 정보, 목표, 현재 과제, 건강 리듬, 이월업무, 오늘 일정, 최근 완료 흐름을 참고합니다.

### Q. 건강 관련 정보는 어떻게 쓰이나요?
의료 진단이 아니라 일정관리 참고용입니다. 운동 강도, 회복 시간, 무리한 일정 여부를 조정하는 데 활용됩니다.

## 15. 운영자 교육용 설명
Beyond Work를 설명할 때는 기능을 나열하기보다 “오늘의 중요한 실행을 놓치지 않게 돕는 플래너”라고 소개하는 것이 좋습니다. 사용자는 모든 기능을 처음부터 쓸 필요가 없습니다. 우선업무와 시간표를 매일 쓰게 만든 뒤, 반복업무, AI 코칭, 프로젝트, 자금관리, 시트 기능을 단계적으로 안내합니다.
