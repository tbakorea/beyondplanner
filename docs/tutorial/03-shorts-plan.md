# Beyond Work 30초 쇼츠 시리즈

## 쇼츠 운영 목적
기능을 짧게 보여주어 사용자가 하나씩 따라 하게 만듭니다. 긴 매뉴얼보다 실제 확산과 학습에 유리합니다.

## 쇼츠 1. 오늘의 우선업무 3개만 정하기
핵심 메시지:
“하루를 바꾸는 것은 긴 목록이 아니라 핵심 3개입니다.”

구성:
- 0:00 업무가 많은 화면
- 0:05 A/B/C 업무 입력
- 0:15 완료 체크
- 0:25 오늘의 완료 숫자 확인

## 쇼츠 2. 반복업무 요일 선택
핵심 메시지:
“매일 할 필요 없는 반복업무는 필요한 요일에만 올리세요.”

구성:
- 반복 우선업무 열기
- 매일 선택
- 일~토 체크박스 중 필요한 요일 선택
- 해당 요일에 자동 생성되는 장면

## 쇼츠 3. 시간표 30분/1시간 전환
핵심 메시지:
“하루가 촘촘하면 30분, 큰 흐름이면 1시간으로 봅니다.”

구성:
- 30분 화면
- 1시간 버튼 클릭
- 00:00~24:00 화면
- 현재 시간대 강조 표시

## 쇼츠 4. AI 추천으로 오늘 할 일 만들기
핵심 메시지:
“AI가 목표, 이월업무, 건강 리듬을 읽고 오늘의 할 일을 제안합니다.”

구성:
- 사용자 정보 입력 화면
- AI 추천 클릭
- 추천 업무 선택
- 우선업무에 자동 추가

## 쇼츠 5. 프로젝트와 자금 이슈를 놓치지 않기
핵심 메시지:
“경영자의 일정은 업무, 프로젝트, 자금 흐름이 함께 움직입니다.”

구성:
- 프로젝트 리스트
- 프로젝트 세부내용
- 자금/용돈 섹션
- 우선업무 반영 예시

## 쇼츠 6. 커스텀 시트 만들기
핵심 메시지:
“필요한 양식은 시트로 바로 만들어 관리합니다.”

구성:
- 시트 목록
- 새 시트 만들기
- 행/열 크기 조정
- 제목행/제목열 고정

## 쇼츠별 공통 포맷
- 첫 2초: 문제 상황 한 문장
- 3-7초: 기능 버튼 또는 입력 화면
- 8-22초: 실제 조작
- 23-28초: 결과 화면
- 29-30초: 짧은 결론

## 공통 CTA 문구
- “오늘의 한 칸부터 시작하세요.”
- “Beyond Work로 실행을 선명하게.”
- “업무와 시간을 한 화면에서 정리하세요.”

## 쇼츠 촬영 팁
- 스마트폰 화면 비율 9:16 기준으로 녹화합니다.
- 손가락 터치가 잘 보이도록 터치 표시를 켭니다.
- 한 영상에는 기능 하나만 보여줍니다.
- 자막은 한 줄 12자 내외로 짧게 유지합니다.
- 배경음악보다 클릭/체크 효과음이 더 잘 어울립니다.

3. 시간표 30분/1시간 전환
4. 반복업무 요일 선택
5. 프로젝트와 자금 이슈
6. 커스텀 시트 만들기
